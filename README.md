# Wanderlust-frontend

I built this project called **Wanderlust-frontend**, a travel-based web application frontend where users can explore different travel destinations.

This project connects with a backend API to fetch destination data and display it in a clean and responsive UI. Users can view all destinations and also see detailed information for each one.

The main goal of this project was to practice frontend development using React and improve my skills in working with APIs, components, and responsive design.

## Tech Stack
- React.js
- JavaScript
- Tailwind CSS / CSS
- API Integration (Fetch/Axios)

## Features
- Display all destinations
- View single destination details
- Responsive design for all devices
